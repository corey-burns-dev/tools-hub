### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/corey-burns-dev/tools-hub.git
   cd tools-hub
   ```

2. Install dependencies:

   ```bash
   bun install
   ```

3. Start the development server:

   ```bash
   bun run dev
   ```
